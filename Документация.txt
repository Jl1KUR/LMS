Установка
Клонируйте репозиторий:

git clone https://github.com/Jl1KUR/LMS.git

Перейдите в директорию проекта:

cd LMS

Запустите приложение на локальном сервере (например, с помощью командной строки).
Примеры запросов
Пример 1: Корректное выражение

curl --location 'http://localhost:8080/api/v1/calculate' \
--header 'Content-Type: application/json' \
--data '{
  "expression": "2+2*2"
}'
Описание: Этот запрос отправляет математическое выражение 2+2*2, и ожидается, что сервер вернет результат вычисления.
Пример 2: Некорректное выражение

curl --location 'http://localhost:8080/api/v1/calculate' \
--header 'Content-Type: application/json' \
--data '{
  "expression": "2a+3"
}'
Описание: Этот запрос отправляет некорректное математическое выражение 2a+3, и сервер должен вернуть сообщение об ошибке.
Ответы сервера
Сервер будет возвращать JSON-ответы, содержащие результат вычисления или сообщение об ошибке в случае некорректного выражения.
Пример ответа на корректный запрос
json
{
  "result": 6
}
Пример ответа на некорректный запрос
json
{
  "error": "Некорректное математическое выражение"
}
